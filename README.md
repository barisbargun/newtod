## <a name="introduction">Introduction</a>

A TODO application that you can add, edit, save, and drag and drop your duties to timetables. You can also see your weekly activities.

## <a name="introduction">Tech Stack</a>

- Vue
- Supabase
- TailwindCSS
- Shadcn Vue

## <a name="features">Features</a>

**Dark and Light Mode**: Quickly and seamlessly switch to dark or light mode.

**Responsive Design**: Ensures optimal display and functionality across all devices.

**Auto Translation**: You can translate the website by clicking button
